# QuotaDEX
An Agent-to-Agent (A2A) secondary market for AI compute. PieBazaar enables AI agents to autonomously buy and sell idle LLM quotas via HTTP 402 interception and Web3 micro-payments on the Kite AI network. | 一个纯粹的 Agent-to-Agent (A2A) AI 算力二级市场。通过 HTTP 402网关拦截与 Kite AI 链上微支付，让 AI Agent 能够全自动地买卖闲置大模型调用额度。
